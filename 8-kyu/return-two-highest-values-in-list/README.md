If there're less than 2 unique values, return as many of them, as possible.</p><p>The result should also be ordered from highest to lowest.</p><p>Examples:</p><pre><code>[4, 10, 10, 9]  =&gt;  [10, 9][1, 1, 1]  =&gt;  [1][]  =&gt;  []</code></pre>
